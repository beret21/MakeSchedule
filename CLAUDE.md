# MakeSchedule - PopClip Extension

## 프로젝트 개요

선택한 텍스트에서 Gemini 2.0 Flash API로 일정 정보를 JSON 구조화 추출하고,
SwiftUI 네이티브 대화상자에서 검수/편집 후 Calendar.app에 EventKit으로 직접 등록하는 PopClip Extension.

## 아키텍처

```text
텍스트 선택 → PopClip ($POPCLIP_TEXT)
→ extract_and_open.sh (Shell 래퍼)
→ gemini_parser.py (Gemini 2.0 Flash → JSON)
→ MakeScheduleUI (SwiftUI 네이티브 앱)
→ EventKit → Calendar.app 등록
```

## 주요 파일

### PopClip Extension (`MakeSchedule.popclipext/`)

- `Config.yaml` — PopClip 설정 (Gemini API Key, network 권한)
- `extract_and_open.sh` — Shell 래퍼 (Python 탐색, 의존성 관리, Gemini 호출, UI 실행)
- `gemini_parser.py` — Gemini API 호출, JSON 구조화 추출
- `MakeScheduleUI` — SwiftUI 바이너리 (빌드 후 복사)
- `icon.svg` — PopClip 아이콘

### SwiftUI 앱 (`MakeScheduleUI/`)

- `Package.swift` — Swift Package Manager 설정 (macOS 13+)
- `Sources/MakeScheduleApp.swift` — 앱 진입점, 윈도우 설정
- `Sources/ScheduleFormView.swift` — 메인 폼 UI (Fantastical 스타일)
- `Sources/CalendarManager.swift` — EventKit 래퍼 (캘린더 목록, 이벤트 생성)
- `Sources/EventInput.swift` — JSON 입력 모델, stdin 리더

## 런타임 의존성

- Python 3.11+ (Gemini API 호출용)
- pip: `google-genai` (첫 실행 시 자동 설치)
- macOS 13+ (SwiftUI, EventKit)
- Gemini API Key (Google AI Studio 무료 발급)

## Gemini JSON 출력 형식

```json
{
  "title": "일정 제목",
  "start_date": "YYYY-MM-DD",
  "start_time": "HH:MM",
  "end_date": "YYYY-MM-DD",
  "end_time": "HH:MM",
  "all_day": false,
  "location": "장소명",
  "notes": "부가 정보 (강의내용, 주소, 링크 등)"
}
```

## 빌드 방법

```bash
cd MakeScheduleUI
swift build -c release
cp .build/release/MakeScheduleUI ../MakeSchedule.popclipext/
open ../MakeSchedule.popclipext  # PopClip 설치
```
